What's New in Version 1.5

The following outlines the highlights of second major release of Joomla.

Important Notes

NOTE: JOOMLA 1.5 IS NOT READY FOR PRODUCTION SITES!

There are several important changes in Joomla 1.5 that you should be aware of. These include:

  • Mambots have been renamed to Plugins.
  • Components, languages, modules, plugins and templates are now collectively known are Extensions.

Menu and Menu Types Manager

  • A new database table has been added to store the menu types, rather than parsing the module parameters and hence increasing performance.
  • Modules with the same menu type can now be added more than once allowing template designers greater flexibility.

 

Modules Manager

  • Added a filter to display modules assigned to templates templates (excludes modules assigned to all templates).

 

Contacts Component

  • Two new fields are added for mobile phone number(s) and a web page.
  • Several database fields have been widened to at least 100 characters to allow more flexibility.
  • Contact position, address and phone numbers now allow for multi-line input.

 

Banners Module

  • Multiple banners can now be shown and style via bannergroup and banneritem css classes.
  • Ordering of the banners can be randomised.

Compatibility

We have made every effort to make this version backwardly compatible with Joomla 1.0 but there are a few changes that may affect a small selection of existing addons.

Please note that addons written for Mambo 4.5.3 and above are probably not compatible with Joomla 1.5.
